# Contributor Missions

Contributor Missions are small, meaningful first contributions to CoSeeker.

They may involve product thinking, UX review, community design, research, documentation, safety, or small code changes.

You do not need to be a senior developer to contribute. Clear thinking, careful observation, and mission alignment are valuable contributions here.

## How to contribute

1. Pick an issue labeled `contributor-mission`.
2. Comment: “I would like to work on this.”
3. Share your contribution as a GitHub issue comment or a small Markdown pull request.
4. Keep the first contribution focused.
5. Maintainers may turn strong contributions into product specs, design tasks, documentation, or implementation issues.
