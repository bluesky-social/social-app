## Run / Test

Install dependencies:

```bash
cd bskyembed
yarn
```

Run the dev server:

```bash
yarn dev
```

You can see the embed homepage at http://localhost:5173

### Testbed

In another terminal window, run the snippet dev script:

```bash
yarn dev-snippet
```

Then, open `file:/<your files>/social-app/bskyembed/test.html` in your browser.
